A- 200: OK – The request was successful.
B- 400: Bad Request – The server could not understand the request because of invalid syntax.
C- 401: Unauthorized – Authentication is required and has failed or not been provided.
D- 403: Forbidden – The server understands the request but refuses to authorize it.
E- 404: Not Found – The requested resource could not be found on the server.
F- 500: Internal Server Error – The server encountered an unexpected condition that prevented it from fulfilling the request.
